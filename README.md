# 💰 AtliQ Hardwares — Excel Financial Analytics Project
 
> **Decoding Profit & Loss across 23 markets, 3 fiscal years, and every quarter — all in Excel.**
 
---
 
## 🚀 Project Overview
 
This repository contains a comprehensive **Financial Analytics solution** built entirely in Microsoft Excel for **AtliQ Hardwares** — a global hardware company operating across 23 markets worldwide.
 
Where sales analytics tells you *how much* you sold, financial analytics tells you *how profitable* you actually were. This project digs deep into:
 
- 📉 Are we making more money — or just making more sales?
- 🌍 Which markets have the healthiest profit margins?
- 📆 How does profitability shift quarter by quarter?
- 🔴 Where are margins shrinking despite revenue growth?
- 🏆 Which regions are the most cost-efficient?
All built with **zero code** — just Excel, Power Query, Power Pivot, DAX, and Conditional Formatting. 💪

## 🛠️ Tools & Techniques Used
 
### 1. 🔄 Power Query — Data Transformation
Raw P&L data from multiple sources was cleaned and structured before loading:
- Normalised fiscal year and month columns
- Unpivoted wide-format data into analysis-ready tables
- Cleaned COGS, Net Sales, and Margin fields across all markets
### 2. 🗃️ Data Model — Connecting the Dots
All tables were brought into Excel's **Data Model** with relationships:
- Linked `fact_financials` with `dim_market`, `dim_date`, and `dim_subzone`
- Enabled Sub-Zone level aggregations without duplicate data
- Star schema design for clean, fast querying
### 3. ⚙️ Power Pivot — Financial Reporting Engine
**Power Pivot** made building dynamic P&L reports possible:
- Created measure tables for clean organisation
- Built multi-level hierarchies (Region → Sub-Zone → Country)
- Enabled YoY comparisons across fiscal years
### 4. 📐 DAX Formulas — Financial Calculations
 
```dax
-- Gross Margin
Gross Margin = [Net Sales] - [Total COGS]
 
-- GM Percentage
GM % = DIVIDE([Gross Margin], [Net Sales], 0)
 
-- Year-over-Year GM% Change
GM % Change = 
DIVIDE(
    [GM % CY] - [GM % PY],
    ABS([GM % PY]),
    0
)
 
-- COGS
Total COGS = SUM(fact_financials[manufacturing_cost]) 
             + SUM(fact_financials[freight_cost])
 
-- Quarterly GM% (Time Intelligence)
GM % Q1 = CALCULATE([GM %], fact_date[quarter] = "Q1")
```
 
### 5. 🎨 Conditional Formatting — Instant Visual Signals
Financial reports use formatting that communicates at a glance:
- 🟢 **Green** for markets beating the global GM% average (36.43%)
- 🔴 **Red** for markets falling below average or declining YoY
- 📊 **Heat maps** on the quarterly GM% table to spot trends instantly
- **Colour-coded variance** columns — negative swings highlighted in red

## 💡 Business Insights & Recommendations
 
1. **Revenue growth is masking a profitability problem** — GM% has fallen from 41.4% → 36.4% over 3 years. COGS is consistently outpacing revenue growth. Cost control needs immediate attention.
2. **India is the elephant in the room** — It's the #1 market by revenue ($161M) but has the lowest GM% of any sub-zone (32%). Fixing India's margins moves the needle more than any other market.
3. **New Zealand & Japan are margin champions** — Despite smaller revenue, their 48% and 46% GM% respectively show what's possible. Study what's working there and replicate it elsewhere.
4. **Germany needs urgent intervention** — 26.18% GM% is the worst of any market. Revenue is growing but it's unprofitable growth.
5. **Q2 (Oct–Dec) is the golden window** — This quarter drives the most revenue consistently. Supply chain and marketing investments timed for Q2 will deliver the highest ROI.
6. **UK is the comeback story** — GM% jumped from 34.1% to 45.1% in one year. Understanding the drivers behind this could unlock similar improvements in struggling markets.
